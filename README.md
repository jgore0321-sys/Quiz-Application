# Quiz-Application
"A Python-based MCQ Quiz Application that allows users to answer multiple-choice questions, tracks scores, and provides instant results through an interactive command-line interface."
## Features
- Multiple Choice Questions
- Score Tracking
- Instant Results

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Run:
   python Quiz.py

## Future Improvements
- GUI using Tkinter
- Timer-based questions
- Question database
