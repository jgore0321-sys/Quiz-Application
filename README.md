# Quiz Application

A simple Python-based Quiz Application that allows users to answer multiple-choice questions, tracks their score, and displays the final result. This project was developed to practice core Python programming concepts such as functions, loops, conditional statements, and user input handling.

## Features

- Multiple Choice Questions (MCQs)
- User-friendly command-line interface
- Instant answer validation
- Score tracking system
- Final result display
- Easy to modify and add new questions

## Technologies Used

- Python 3

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/jgore0321-sys/Quiz-Application.git
```

2. Navigate to the project folder:

```bash
cd Quiz-Application
```

3. Run the program:

```bash
python Quiz.py
```
